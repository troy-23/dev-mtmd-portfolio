# Context

## What this is

The personal portfolio website of **Marlone Troy Dominguiano** — a graduating
AI-assisted full-stack developer, AI content creator, and freelance web developer.

It is a single-page marketing/portfolio site whose job is to win freelance and
employment opportunities by showcasing shipped work and a clear value proposition.

## Positioning / message

The core message the site must communicate:

- **Skilled at using and utilizing AI** (agentic coding, prompt engineering) —
  while **understanding the fundamentals** (how the web works, clean code, data
  modeling, debugging).
- **Solves real-world problems** and builds around **what the client actually needs**,
  not just what's technically interesting.
- **Stack-flexible** — not locked to React/Next/TypeScript; AI makes it practical to
  pick the right tools per project.
- **Ships fast and reliably** — landing pages in ~4–7 days, full builds in ~20+ days,
  timelines flexing to the client's goals.

> Tone guardrail: never make it sound like "all AI, no original ideas." AI is a tool on
> top of real fundamentals and judgment. Avoid unverifiable claims (e.g. "100%
> satisfaction", hard time guarantees like "48 hours").

## Audience
